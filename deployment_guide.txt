# G1axWorldGenerator Deployment Guide

## 1. GitHub Push

### First time setup:
```bash
cd /mnt/d/CustomPlugins/G1axWorldGenerator
git init
git add .
git commit -m "Initial commit - G1axWorldGenerator v1.1 with 9 biomes and optimizations"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/G1axWorldGenerator.git
git push -u origin main
```

### For updates:
```bash
git add .
git commit -m "v1.1 - Added End/Nether/Snow/Ice/Void biomes + performance optimizations"
git push origin main
```

### Create Release:
1. Go to GitHub repository
2. Click "Releases" → "Create a new release"
3. Tag: v1.1
4. Title: "G1axWorldGenerator v1.1 - 9 Biomes + Optimizations"
5. Upload: target/G1axWorldGenerator.jar
6. Description: See changelog below

## 2. Build the Plugin

```bash
# Make sure you have Java and Maven installed
cd /mnt/d/CustomPlugins/G1axWorldGenerator
mvn clean package
```
The JAR will be in: target/G1axWorldGenerator.jar

## 3. SpigotMC Upload

### Requirements:
- SpigotMC account
- Plugin must be tested on Spigot server

### Steps:
1. Go to: https://www.spigotmc.org/resources/
2. Click "Post a resource"
3. Fill out form:
   - Name: G1axWorldGenerator
   - Tag line: "Custom world generator with 9 optimized biomes"
   - Category: World Generators
   - Upload JAR file
   - Add screenshots of different biomes
   - Description: Use changelog below

## 4. Modrinth Upload

### Steps:
1. Go to: https://modrinth.com/dashboard/projects
2. Click "Create a project"
3. Fill out:
   - Name: G1axWorldGenerator
   - Summary: "Performance-optimized world generator with 9 unique biomes"

   - Supported versions: 1.19+, 1.20+
   - Description: Use changelog below

## 5. Version Changelog Template

### G1axWorldGenerator v1.1

**New Features:**
- Added 5 new biome themes: END, NETHER, SNOW, ICE, VOID
- Comprehensive world optimizations for maximum performance
- Automatic world configuration on load

**Biomes Available (9 total):**
- PLAINS - Grass terrain with oak trees
- DESERT - Sand terrain with cacti
- BADLANDS - Mesa/terracotta terrain  
- MUSHROOM - Mushroom island terrain
- END - End stone terrain with chorus plants
- NETHER - Netherrack terrain with crimson trees
- SNOW - Grass terrain with snow layers and spruce trees
- ICE - Frozen terrain with sharp ice spikes
- VOID - Floating islands in void space

**Performance Optimizations:**
- Disabled mob spawning, weather, day/night cycles
- Reduced view/simulation distance
- Disabled auto-save, fire spread, random ticks
- 1000 block world border
- Fixed daytime and clear weather

**Usage:**
```
/mv create world_name normal -g G1axWorldGenerator:BIOME_TYPE
```

**Requirements:**
- Bukkit/Spigot 1.19+
- Multiverse-Core (recommended)

## 6. Update Version Numbers

Before releasing, update version in:
- pom.xml (line with <version>1.0</version> → <version>1.1</version>)
- plugin.yml (version: 1.0 → version: 1.1)

## 7. Testing Checklist

Before uploading:
- [ ] Plugin compiles without errors
- [ ] All 9 biomes generate correctly
- [ ] World optimizations apply automatically
- [ ] No console errors on world creation
- [ ] Multiverse commands work properly
